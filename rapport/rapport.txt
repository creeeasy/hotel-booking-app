Fatiel est une application mobile.
C'est une plateforme dédiée à l'hôtellerie en Algérie.

Dans cette application, il y a deux types d'utilisateurs : Visiteur et Hôtel.
Ils ont des interfaces d'inscription différentes mais partagent la même interface d'authentification.

Authentification
L'authentification comprend :

Page d'inscription :

Visiteur : Pour s'inscrire, le visiteur doit renseigner son email, un mot de passe, une confirmation du mot de passe et son prénom.
Ensuite, il a la possibilité de choisir une localisation (ce champ est optionnel).
Hôtel : Pour s'inscrire, l'hôtel doit renseigner son email, le nom de l'hôtel, sa localisation et une description.
Autres interfaces :

Mot de passe oublié : Permet de réinitialiser le mot de passe.
Vérification de l'email : Processus de confirmation après l'inscription.
Interface du Visiteur
Une fois connecté, le visiteur accède à une page d’exploration qui fait office de page d’accueil.
La navigation inférieure comporte trois sections :

Explorer
Réservations
Favoris
1. Explorer
La première page de l'exploration permet au visiteur de rechercher un hôtel par son nom.
On y trouve deux onglets :

Hôtels recommandés avec un bouton "Voir tout".
Hôtels à proximité avec un bouton "Voir tout".
Il peut aussi découvrir les villes disponibles via un carrousel des 58 wilayas algériennes.
En cliquant sur une wilaya, la page affiche tous les hôtels de cette région.

Détails d’un hôtel
Si l’utilisateur clique sur une carte d’hôtel, il sera redirigé vers la page de détails de l'hôtel, où il pourra voir :

Nom de l'hôtel
Description
Images
Évaluations et notes
Vignette (thumbnail)
Localisation
Informations de contact
Voir l'hôtel sur Google Maps
Explorer les offres de chambres
Sur cette page, il peut également ajouter l'hôtel à ses favoris pour le retrouver plus tard.

En cliquant sur "Voir tous les avis", il sera redirigé vers la page des avis de l'hôtel, qui affiche les évaluations des utilisateurs avec leurs commentaires et leurs notes.
En cliquant sur "Ouvrir dans Google Maps", il sera redirigé vers Google Maps.
En cliquant sur "Voir les offres de chambres", il sera dirigé vers la page correspondante.
Explorer les offres de chambres
Dans cette section, le visiteur peut choisir la chambre qui lui convient.
Chaque chambre possède plusieurs attributs :

Nom
Description
Capacité
Prix par nuit
Équipements
Disponibilité :
Soit l'hôtel indique qu’elle est disponible.
Soit elle est réservée, avec la prochaine date disponible affichée.
Images
Si le visiteur clique sur "Réserver maintenant", il pourra :

Sélectionner la durée de son séjour.
Confirmer sa réservation.
2. Réservations
Cette section comporte trois onglets :

Réservations en attente

Réservations confirmées

Réservations annulées

Réservations en attente : L'hôtel n’a pas encore confirmé la réservation. Le visiteur peut voir les détails de sa réservation et l’annuler si nécessaire.

Réservations confirmées : Liste des réservations complétées avec un message indiquant que le séjour est terminé.

Réservations annulées : Liste des réservations annulées avec un message approprié.

3. Favoris
Dans cet onglet, l’utilisateur peut retrouver les hôtels qu’il a ajoutés en favoris.
